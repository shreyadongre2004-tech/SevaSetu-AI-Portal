# SevaSetu AI

SevaSetu AI is a smart web application that simplifies government form filling using Optical Character Recognition (OCR). Citizens often need to repeatedly enter personal details such as name, date of birth, address, and identification numbers while applying for government services. This project automates that process by extracting information directly from uploaded documents.

Users can upload documents such as Aadhaar Cards, PAN Cards, certificates, and other identity proofs in image or PDF format. The system processes these documents using OCR technology and automatically extracts important details. The extracted information is then used to pre-fill application forms, reducing manual effort and minimizing errors.

A user-friendly dashboard allows users to review, verify, and edit the extracted data before generating the final form. The platform supports multiple Indian languages, including English, Hindi, and Marathi, making it accessible to a wide range of users.

### Key Features

* OCR-based document data extraction
* Support for image and PDF uploads
* Automatic form pre-filling
* Editable verification dashboard
* Multi-language support
* Downloadable completed forms

SevaSetu AI aims to improve efficiency, accuracy, and accessibility in government services while supporting the vision of Digital India through technology-driven governance.
